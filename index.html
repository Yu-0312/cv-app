雲端與登入

正式上線時，每位使用者用自己的 Google 帳號登入，只能讀寫自己的 CV。

Supabase 設定
這個頁面使用 Supabase 的 Google OAuth redirect flow，不使用已淘汰的 `platform.js` / `g-signin2`。請在 Supabase Authentication 內啟用 Google Provider，並把 Site URL 與 redirect URL 指向你的網站。
登入偵錯資訊
Google 登入影響評估
Google 舊版 Sign-In 程式庫已淘汰，且 FedCM 已成為必要方向。這個專案目前走 Supabase OAuth redirect，不直接依賴舊版 Google Sign-In JavaScript client。

專業技能與亮點

工作 / 實習經歷

每一筆請用 `職稱 | 單位 | 日期 | 內容`，空一行分隔下一筆。

學歷

每一筆請用 `學校 / 科系 | 學位或類型 | 日期 | 說明`，空一行分隔。

專案 / 作品

每一筆請用 `專案名 | 類型或技術 | 日期 | 內容`，空一行分隔。

獎項 / 證照
